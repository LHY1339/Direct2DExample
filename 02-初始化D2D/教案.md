## 创建 ID2D1Factory
- 创建变量
```
ID2D1Factory* MainID2D1Factory = nullptr;
```
- 初始化
```
D2D1CreateFactory(D2D1_FACTORY_TYPE_MULTI_THREADED, &MainID2D1Factory);			//创建D2D工厂
```
## 创建 ID2D1HwndRenderTarget
- 变量
```
ID2D1HwndRenderTarget* MainID2D1HwndRenderTarget = nullptr;
```
- 初始化
```
RECT wnd_rect = {};																												//创建 RECT 结构体
GetWindowRect(MyWindow, &wnd_rect);																								//获取窗口RECT
const D2D1_SIZE_U wnd_size = D2D1::SizeU(wnd_rect.right, wnd_rect.bottom);														//转换为 D2D1_SIZE_U
const D2D1_RENDER_TARGET_PROPERTIES render_target_property = D2D1::RenderTargetProperties();									//创建 D2D1_RENDER_TARGET_PROPERTIES
const D2D1_HWND_RENDER_TARGET_PROPERTIES hwnd_render_target_property = D2D1::HwndRenderTargetProperties(MyWindow, wnd_size);	//创建 D2D1_HWND_RENDER_TARGET_PROPERTIES
MainID2D1Factory->CreateHwndRenderTarget(render_target_property, hwnd_render_target_property, &MainID2D1HwndRenderTarget);		//创建 HwndRenderTarget，填入创建的变量
```
## 设置背景为蓝色
- 绘制背景
```
MainID2D1HwndRenderTarget->BeginDraw();										//开始绘制
const D2D1_COLOR_F clear_color = D2D1::ColorF(0.0f, 0.6f, 1.0f, 1.0f);		//创建 D2D1_COLOR_F 并且设置为蓝色
MainID2D1HwndRenderTarget->Clear(clear_color);								//清除背景为蓝色
MainID2D1HwndRenderTarget->EndDraw();										//结束绘制
```
