## 注册窗口类
- 创建过程函数
```
LRESULT CALLBACK MainWindowProc(HWND hwnd, UINT uMsg, WPARAM wParam, LPARAM lParam)
{
	return DefWindowProcW(hwnd, uMsg, wParam, lParam);
}
```
- 创建一个 WNDCLASS 结构体
```
WNDCLASS wnd_class = {};
```
- 设置 WNDCLASS
```
wnd_class.lpfnWndProc = MainWindowProc;                 //过程函数
wnd_class.lpszClassName = L"MyWindowClass";             //类名
wnd_class.hInstance = GetModuleHandle(nullptr);         //进程句柄
wnd_class.hIcon = LoadIcon(nullptr, IDI_APPLICATION); 	//图标
wnd_class.hCursor = LoadCursor(nullptr, IDC_ARROW);  	//鼠标样式
```
- 注册 WNDCLASS
```
RegisterClass(&wnd_class)
```
## 创建窗口并显示
- 创建窗口
```
HWND MyWindow;
MyWindow = CreateWindowEx(
	0,                          //样式
	L"MyWindowClass",           //类名
	L"MyWindowTitle",    	    //标题名
	WS_ACTIVECAPTION,  			//类型
	CW_USEDEFAULT,              //X位置
    CW_USEDEFAULT,              //Y位置
	600,                        //宽度
    400,                        //高度
	nullptr,                    //父窗口
    nullptr,                    //菜单和标识
	GetModuleHandle(nullptr),   //进程句柄
	nullptr                     //数据指针
);
```
- 显示窗口
```
ShowWindow(MyWindow, true);
```
## 消息循环
- 创建 MSG 结构体
```
MSG msg = {};
```
- 循环获取并传递消息
```
while (GetMessageW(&msg, NULL, 0, 0)) 
{
	TranslateMessage(&msg);             //获取消息
	DispatchMessageW(&msg);             //传递消息给窗口
}
```
